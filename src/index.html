PRUEBA DE RESPONSIVIDAD




Las pruebas de estrés de software son comúnmente realizadas en etapas avanzadas, cuando los distintos módulos que conforman al sistema han sido integrados. La importancia de estas pruebas recae en que pueden ayudarte a mejorar la calidad de tus productos informáticos sin poner en riesgo los recursos invertidos.
Breakpoint: Es un punto de interrupción en la ejecución del proceso o programa en ejecución. Los Breakpoints permiten especificar detenciones del programa, para observar valores de memoria, registros o variables durante la ejecución. Existen varias formas alternativas de activar un punto de interrupción, Breakpoint.
Esta es una pagina de prueba para verificar el diseño responsivo en la cual se utilizó media query asi como tambien una adaptacion a las etiquetas en css usando una flex-box.
  • Media query: es una técnica de CSS utilizada para aplicar estilos condicionalmente, dependiendo de las características del dispositivo, como el tamaño de la pantalla, la orientación, la resolución, entre otros.
  • Flex-box: Se utilizo Flexbox para diseñar de manera responsiva un parte de la pagina la cual se encuentra en el contenedor 'home' al cual se le aplico la propiedad 'display: flex' la cual convirtió a dicho contenedor en un contenedor flexible.
